# React Todo App

A simple and fully functional Todo application built using React.  
Designed to demonstrate frontend fundamentals, state management, and local storage.

## Features
- Add, delete, and complete todos
- Persistent data using LocalStorage
- Responsive and clean UI
- Keyboard support (Enter to add tasks)

## Tech Stack
- React
- JavaScript (ES6)
- CSS

## Getting Started
```bash
npm install
npm run dev
